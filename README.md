# home-manager

## Steps
- Run `ssh-keygen -t ed25519`
- Add public key to git
- Run `cd ~/.config`
- Run `sudo apt install git -y && git clone git@github.com:EvanMusic14/home-manager.git`
- Run `cd home-manager`
- Run `./bin/setup`
